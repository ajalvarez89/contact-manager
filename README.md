# Contact Manager

This project help us to upload a contact list from .csv file.

## Setup

Just run the bin setup:

```bash
  bin/setup
```
or

```bash
  bundle install
  rails db:setup
  yarn install --check-files
```

To run the app:

```bash
  cp .env.example .env
  source .env
  and
  rails server
```
